# Sistema de Donaciones de Frutas y Verduras

Aplicación web que permite registrar donaciones y solicitudes de frutas o verduras con el objetivo de fomentar la solidaridad y evitar la pérdida de alimentos provenientes de cultivos domésticos.

## Contexto del proyecto

Este proyecto fue desarrollado originalmente como parte de una tarea de Desarrollo de Aplicaciones Web.  
La premisa consistía en crear una aplicación web que permitiera a las personas informar la disponibilidad de frutas o verduras para donar, así como registrar pedidos cuando necesitaran algún alimento.

Posteriormente se realizaron mejoras al proyecto original, incluyendo cambios en la estructura del código, la interfaz y la visualización de datos. Sin embargo, el sistema aún puede seguir desarrollándose y presenta varias oportunidades de mejora.

## Funcionalidades

- Registro de donaciones de frutas o verduras
- Registro de pedidos de frutas o verduras
- Visualización de donaciones y pedidos disponibles, mediante tablas y mapa
- Estadísticas mediante gráficos
- Interfaz web para interactuar con el sistema
